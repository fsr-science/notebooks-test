**Q51.** Show that $A \cup B = A \cap B$ implies $A = B$.

> [!example] Solution
> Let $a \in A$. Then $a \in A \cup B$. Since $A \cup B = A \cap B$, we have $a \in A \cap B$, so $a \in B$. Thus $A \subseteq B$.
>
> Similarly, let $b \in B$. Then $b \in A \cup B = A \cap B$, so $b \in A$. Thus $B \subseteq A$.
>
> Since $A \subseteq B$ and $B \subseteq A$, we conclude $A = B$.

**Q52.** For any sets $A$ and $B$, show that $A = (A \cap B) \cup (A - B)$.

> [!example] Solution
> Let $x \in A$. Either $x \in B$ or $x \notin B$.
>
> Case 1: $x \in B$ тАФ then $x \in A \cap B \subseteq (A \cap B) \cup (A - B)$
>
> Case 2: $x \notin B$ тАФ then $x \in A$ and $x \notin B$, so $x \in A - B \subseteq (A \cap B) \cup (A - B)$
>
> Thus $A \subseteq (A \cap B) \cup (A - B)$.
>
> Conversely, every element of $(A \cap B) \cup (A - B)$ is in $A$ (since both $A \cap B \subseteq A$ and $A - B \subseteq A$). So $(A \cap B) \cup (A - B) \subseteq A$.
>
> Therefore $A = (A \cap B) \cup (A - B)$.

**Q53.** Prove that $(A \cup B)' = A' \cap B'$ using the definition of complement.

> [!example] Solution
> Let $x \in (A \cup B)'$.
>
> $\Rightarrow x \notin A \cup B$
>
> $\Rightarrow x \notin A$ and $x \notin B$
>
> $\Rightarrow x \in A'$ and $x \in B'$
>
> $\Rightarrow x \in A' \cap B'$
>
> So $(A \cup B)' \subseteq A' \cap B'$.
>
> Conversely, if $x \in A' \cap B'$, then $x \notin A$ and $x \notin B$, so $x \notin A \cup B$, meaning $x \in (A \cup B)'$.
>
> So $A' \cap B' \subseteq (A \cup B)'$.
>
> Therefore $(A \cup B)' = A' \cap B'$.

**Q54.** Show that if $A \subseteq B$, then $C - B \subseteq C - A$.

**Q60.** Show that the four conditions are equivalent: $A \subseteq B \iff A - B = \phi \iff A \cup B = B \iff A \cap B = A$.

> [!example] Solution
> **(i) $A \subseteq B \Rightarrow A - B = \phi$:** If $A \subseteq B$, every $x \in A$ is in $B$, so no $x \in A$ with $x \notin B$. Hence $A - B = \phi$.
>
> **(ii) $A - B = \phi \Rightarrow A \cup B = B$:** Since $A - B = \phi$, all elements of $A$ are in $B$, so $A \subseteq B$. Thus $A \cup B = B$.
>
> **(iii) $A \cup B = B \Rightarrow A \cap B = A$:** $A = A \cap (A \cup B) = A \cap B$.
>
> **(iv) $A \cap B = A \Rightarrow A \subseteq B$:** If $x \in A = A \cap B$, then $x \in B$. So $A \subseteq B$. тЬЕ

**Q62.** For three sets $A$, $B$, $C$ prove that $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$.

> [!example] Solution
> Let $x \in A \cap (B \cup C)$. Then $x \in A$ and $x \in B \cup C$.
>
> Case 1: $x \in B$ тАФ then $x \in A \cap B$
>
> Case 2: $x \in C$ тАФ then $x \in A \cap C$
>
> Either way, $x \in (A \cap B) \cup (A \cap C)$.
>
> Conversely, if $x \in (A \cap B) \cup (A \cap C)$, then $x \in A$ and ($x \in B$ or $x \in C$), so $x \in A \cap (B \cup C)$.
>
> Therefore equality holds.

**Q63.** Show that $A \cup B = A \cup C$ and $A \cap B = A \cap C$ together imply $B = C$.

>
> Similarly, $C = C \cap (A \cup C) = C \cap (A \cup B) = (A \cap C) \cup (B \cap C)$
>

**Q64.** In a town of 10,000 families, it was found that 40% families buy newspaper A, 20% buy B, 10% buy C, 5% buy A and B, 3% buy B and C, 4% buy A and C, and 2% buy all three. Find the number of families that buy newspaper A only.
